# Flaretion - 智能策略分析平台

Flaretion 是一个基于 Ionic React 的智能策略分析平台移动端应用，专注于多周期RSI共振分析和技术指标组合，为交易者提供精准的市场状态判断和智能信号提醒。

## 🚀 功能特性

### 核心功能
- **多周期RSI共振分析** - 6/12/24小时RSI组合判断，识别抄底逃顶机会
- **智能市场状态识别** - 横盘整理、趋势上涨、趋势下跌、趋势反转、不确定状态
- **模块化策略库** - 傻瓜模式、自由模式、AI优化模式
- **实时交易信号** - 基于多维度数据的智能信号生成
- **策略管理系统** - 创建、编辑、监控交易策略

### 差异化优势
- **技术分析专业化** - 专注于策略分析而非交易执行
- **中国市场特色** - 中文界面、本土化策略、中国交易所集成
- **模块化设计** - 可组合的指标和条件系统
- **AI参数优化** - 自动调整策略参数和阈值


## 🛠️ 技术栈

- **前端框架**: Ionic React + TypeScript
- **UI组件**: Ionic Framework + Ionicons
- **状态管理**: React Hooks
- **HTTP客户端**: Fetch API
- **样式**: CSS3 + Ionic CSS Variables
- **构建工具**: Vite
- **PWA支持**: 渐进式Web应用
- **部署平台**: Netlify

## 📦 安装和运行

### 前置要求
- Node.js 20+ (推荐使用 Node.js 20 以获得最佳兼容性)
- npm 10+ 或 yarn
- 现代浏览器支持

```bash
npm install
```


## 📱 应用页面

### 主要功能页面

- **📊 策略** - 策略管理、创建、编辑、分类筛选

- **RSI共振分析** - RSI6/12/24数值和共振状态判断
- **市场状态识别** - 综合得分、状态分类、置信度分析
- **多周期权重** - 不同时间周期的重要性权重显示

## 🔌 后端连接配置

### API端点
前端默认连接到 `http://localhost:8000`，支持以下API端点：

- `GET /api/klines` - 获取K线数据
- `GET /api/signals` - 获取交易信号
- `GET /api/market-state` - 获取市场状态
- `GET /api/strategies` - 获取策略数据
- `GET /api/analysis` - 获取分析结果
- `GET /api/status` - 获取系统状态
- `GET /api/health` - 健康检查

### 修改API地址
1. 在个人中心设置页面中修改"API基础URL"
2. 或直接修改 `src/services/api.ts` 中的 `API_BASE_URL` 常量

## 🔧 开发指南

### 项目结构
```
src/
├── pages/              # 页面组件
│   ├── HomePage.tsx    # 首页
│   ├── StrategyPage.tsx # 策略页面
│   ├── AnalysisPage.tsx # 分析页面
│   ├── SignalsPage.tsx  # 信号页面
│   └── ProfilePage.tsx  # 个人中心
├── services/            # API服务
│   └── api.ts          # API接口定义
├── theme/               # 主题配置
└── App.tsx             # 主应用组件
```

### 添加新页面
1. 在 `src/pages/` 目录下创建新的页面组件
2. 在 `App.tsx` 中添加路由和标签页
3. 更新导航图标和标签

### 自定义样式
- 使用 Ionic CSS Variables 进行主题定制
- 在对应页面的 `.css` 文件中添加自定义样式
- 支持响应式设计和暗色模式

## 🚨 故障排除

### 常见问题

#### 1. 无法连接到后端
- 检查后端服务是否运行
- 验证API地址是否正确
- 确认CORS配置是否正确

#### 2. 数据不显示
- 检查网络请求是否成功
- 查看浏览器控制台错误信息
- 验证API响应格式是否正确

#### 3. 样式显示异常
- 清除浏览器缓存
- 检查CSS文件是否正确加载
- 验证Ionic版本兼容性

### 调试模式
在设置页面启用"调试模式"可以：
- 显示详细的网络请求信息
- 输出API响应数据
- 记录用户操作日志
